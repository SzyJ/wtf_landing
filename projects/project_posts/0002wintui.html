WinTUI

Customisable user interface for C++ Windows terminal programs!

Basic Idea

This header-only C++ library lets you create simple menu selectors and input prompts that are controllable with arrow keys. There are 16 different colous that can be used to customise how the UI looks and feels!

Usage

Simply provide implementations for lambda functions that will be called when drawing the UI. Different widgets will have different customisable visuals and logic. Take a look at the GitHub repo if you would like to see how simple it is to customise!
